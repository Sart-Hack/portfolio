Portfolio Website Spec - Sarthak Gupta
Context
Sarthak is an AI Innovation Specialist at IgniteTech (also working with Khoros and Voriq AI), studying CS at Delhi Technological University. He wants an impressive, scroll-driven 3D portfolio website inspired by oryzo.ai's cinematic scroll experience, with a futuristic/techy aesthetic.

Design Direction: Futuristic / Techy
Background: Deep black/dark navy (#0a0a0f) with subtle grid lines or dot matrix pattern
Accent colors: Electric cyan (#00f0ff), neon purple (#8b5cf6), with subtle green (#22c55e) highlights
Typography: Monospace for headers (JetBrains Mono or Space Mono), clean sans-serif for body (Inter)
Visual effects: Scanline overlays, glowing borders, particle backgrounds, terminal-style text reveals
Overall feel: Like a high-tech command center / AI dashboard aesthetic
Sections & Scroll Flow
The entire page is one continuous scroll experience. Each section occupies ~100vh and transitions are driven by GSAP ScrollTrigger scrubbing.

3D floating laptop/desk scene centered
Name "SARTHAK GUPTA" in large monospace type, glitch/terminal-style reveal animation
Subtitle: "AI Innovation Specialist" with a typing/cursor effect

Small scroll indicator arrow at bottom with pulse animation

A stylized low-poly or sci-fi laptop model floating in space

3D Model Details
Laptop/Desk Scene
Source: Free GLTF model from Sketchfab (e.g., "Sci-fi laptop", "Cyberpunk workstation") or create a simple one in Spline
Fallback: If no good free model exists, use a simple procedural geometry (box + plane for screen) with emissive materials and glowing edges
Lighting: Single directional light + cyan point light from the "screen" + subtle ambient
Environment: Use drei's <Environment> with a dark HDRI for reflections
Post-processing: Optional bloom effect on the screen glow (via @react-three/postprocessing)
Background Elements
Floating particles (drei's <Stars> or custom particle system)
Grid floor that fades into the distance
Optional: scanning line effect (CSS overlay)
Performance Considerations
Lazy load 3D scene: Use React.lazy + Suspense with a loading screen
Model optimization: Keep GLTF under 2MB, use Draco compression
Mobile: Reduce particle count, simplify 3D scene, fallback to CSS animations if WebGL not supported
Prefers-reduced-motion: Respect user preference, disable scroll animations
